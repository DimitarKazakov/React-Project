# :earth_americas: About this Web-Project
**Junk Shop Website** is my defense project for **ReactJs** course at [SoftUni](https://softuni.bg/ "SoftUni") (February-April 2021). The project is **SPA ReactJs Application** with a small Asp.Net Core Web API Backend.

# :evergreen_tree: Overview
**Junk Shop Website** is a website like CraigList or [Olx](https://www.olx.bg).Shop like website for reselling second hand products(services). Users can add product/service, create account, contact users if they are interested in particular product/service, browse them and order them as they like also small search to help them while browsing. They can access theirs and other users profiles. They can see what certain user liked, wish listed and added.
